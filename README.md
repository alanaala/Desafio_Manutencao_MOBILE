# Controle de Gastos - Aplicação Mobile (Android)

## Descrição  
App Android para controle simples de gastos, permitindo adicionar descrições e valores decimais. Exibe lista de gastos e total acumulado.

## Tecnologias  
- Java  
- Android Studio  
- ListView e ArrayAdapter

## Funcionalidades  
- Adicionar gastos com validação de campos.  
- Exibir lista dinâmica de gastos.  
- Cálculo do total atualizado em tempo real.  
- Remover gastos via clique longo.  
- Tratamento de erros para entradas inválidas.

## Como usar  
1. Abra o app no dispositivo/emulador.  
2. Insira descrição e valor do gasto.  
3. Clique em "Adicionar Gasto".  
4. Para remover, toque longo sobre o item da lista.
